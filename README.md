# Mohamed Ait‑Oufquir Portfolio

A modern, beautiful, and slightly futuristic personal portfolio website for a Software Engineer / Data & QA.

 Features
- Hero/Intro: Bold greeting and dynamic titles.
- About Me: Professional bio and experience summary.
- Skills: Categorized technical skills with interactive chips.
- Projects: Grid of project cards with tech stack and links.
- Experience: Vertical timeline of professional history.
- Contact Form: Fully functional contact form connected to a Node.js backend.
- Design: Deep black theme with Cyan accents, glassmorphism, and smooth animations.
- Responsive: Optimized for desktop, tablet, and mobile.

Tech Stack
- Frontend: React, TypeScript, Vite, Vanilla CSS, Lucide React.
- Backend: Node.js, Express, Nodemailer, CORS.

 Getting Started

Prerequisites
- Node.js (v14 or higher)
- npm
